DeepLearning_ImageClassification_Binary

Project Overview

This project is a binary image classification solution for detecting bridge deck damage using deep learning and ML.NET. The model is trained using transfer learning, which adapts a pretrained ResNet model for the specific task.

How It Works

The process starts with loading images from the dataset, followed by preprocessing them into a suitable format. The model is then trained and evaluated to ensure accuracy. Finally, predictions are made on new images, and the results are displayed.

Code Analysis

The code defines several functions to load, preprocess, train, and evaluate the model. The `LoadImagesFromDirectory` function reads images and labels from the dataset. The `ClassifySingleImage` and `ClassifyImages` functions demonstrate the model's capability to predict the class of new images.
